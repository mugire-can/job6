Title: Push to Remote
Category: Git Operations
Difficulty: Easy
Time Estimate: 1 minute

================================================================================
DESCRIPTION
================================================================================

Push your local commits to the remote repository (GitHub) to share your
changes with others.

================================================================================
COMMANDS
================================================================================

Push current branch:
────────────────────────────────────────────────────────────────────────────
git push

Push to specific remote and branch:
────────────────────────────────────────────────────────────────────────────
git push origin development
git push origin main

Push with upstream tracking:
────────────────────────────────────────────────────────────────────────────
git push -u origin development

Push all branches:
────────────────────────────────────────────────────────────────────────────
git push --all

================================================================================
WHAT THIS DOES
================================================================================

1. Uploads your local commits to GitHub
2. Updates the remote branch
3. Makes your commits visible to others
4. -u flag sets up tracking if new branch
